🌐 Flask API Endpoints

GET
/api/battery/cells

Purpose: Get latest voltage and SoC for all battery cells
Response: JSON array with battery_id, avg_voltage, avg_soc

GET
/api/battery/temperature

Purpose: Get recent temperature readings for charts
Response: JSON array with last 20 temperature values

GET
/api/battery/status/{battery_id}

Purpose: Get detailed status for specific battery
Response: Current cycle, operation_type, alarms

GET
/api/battery/history/{battery_id}

Purpose: Get historical data for D3.js charts
Response: Time-series data for voltage, current, temp

✨ Key Features

🔋 Real-time Battery Monitoring

Live voltage, current, and SoC tracking for each cell with color-coded status indicators.

🌡️ Temperature Management

Temperature sensors monitoring with D3.js charts and thermal alerts for safety.

⚠️ Intelligent Alarm System

Automated fault detection with visual and status-based alarm notifications.

📊 Data Visualization

Interactive D3.js charts for historical trends and real-time data analysis.

🔄 Cycle Tracking

Charge/discharge cycle monitoring with performance degradation analysis.

🎯 Cell Balance Monitoring

Individual cell voltage tracking to detect imbalances and optimize performance.
